│ 📋 SETUP CHECKLIST                                                          │
└─────────────────────────────────────────────────────────────────────────────┘

PHASE 1: Google Sheets Setup
   □ Create Google Sheet (sheets.google.com)
   □ Copy Sheet ID from URL
   □ Create Google Apps Script (script.google.com)
   □ Copy code from GOOGLE_APPSCRIPT_CODE.gs
   □ Replace SPREADSHEET_ID with your ID
   □ Run initializeSheet() function
   □ Check sheet has headers

PHASE 2: Deploy AppScript
   □ Click Deploy → New Deployment
   □ Select Type: "Web app"
   □ Execute as: "Me"
   □ Allow: "Anyone"
   □ Deploy
   □ Copy deployment URL
   □ Run testAppScript() to verify

PHASE 3: Configure Your App
   □ Open .env.local
   □ Add GOOGLE_APPSCRIPT_DEPLOYMENT_URL
   □ Run database migration (SQL file)
   □ Restart app: npm run dev

PHASE 4: Test Everything
   □ Add new resource from app
   □ Resource appears locally ✓
   □ Resource appears in Google Sheet ✓
   □ Type filter works ✓
   □ Feedback works ✓
   □ Delete works ✓
